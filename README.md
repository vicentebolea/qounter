# INSTALL

1. Install Qt:
  - Ubuntu/Debians
```
     sudo apt-get install qt5-default pkg-config qttools5-dev-tools
```
  - OpenSUSE
```
     sudo zypper in -t pattern devel_qt5 pkg-config
```
2. Compile and Install
```
    make && sudo make install
```
